## üî¨ Technical Specifications

### Machine Learning Model

**Architecture:** Logistic Regression + TF-IDF Vectorization

**Performance Metrics:**
- üéØ **100% Recall** - Catches ALL phishing URLs (zero false negatives)
- ‚úÖ **93.3% Precision** - Minimal false positives on legitimate sites
- ‚ö° **69.6 URLs/second** - Real-time processing speed
- üìä **99.93% Overall Accuracy** - Production-ready performance

**Training Data:**
- 549,346 URLs from Kaggle phishing dataset
- Binary classification (phishing vs. legitimate)
- Regular model retraining for emerging threats

**Feature Extraction:**
- URL structure analysis (length, special characters, domain)
- TF-IDF text vectorization
- Domain reputation signals
- HTTPS/HTTP analysis

### Backend Architecture

**Framework:** FastAPI (Python 3.11+)

**Key Components:**
- üîê **Authentication** - JWT-based user auth
- ü§ñ **ML Service** - Model inference engine
- üìÅ **File Upload** - CSV processing with validation
- üìä **Analytics Engine** - Statistics aggregation
- üóÑÔ∏è **Database** - SQLite (dev) / PostgreSQL (prod)
- üì° **REST API** - OpenAPI-documented endpoints
- ‚ö° **Celery Workers** - Background task processing
- üíæ **Redis** - Caching & task queue

**Performance Optimizations:**
- ThreadPoolExecutor for parallel bulk scanning (5.9x speedup)
- Result caching for repeated URL checks
- Async/await for concurrent operations
- Connection pooling for database

**Framework:** React 18 + TypeScript + Vite

**Key Technologies:**

### Quick Start

**1. Clone Repository**
```bash
git clone https://github.com/yourusername/phishguard.git
cd phishguard
```

**2. Start Backend**
```bash
cd backend
pip install -r requirements.txt
python main.py
# API runs on http://localhost:8000
```

**3. Start Frontend**
```bash
cd ..
npm install
npm run dev
# Web app runs on http://localhost:8080
```

**4. Load Extension**
```bash

# 2. Go to chrome://extensions/

# 4. Click "Load unpacked"
```

**5. Test Everything**
```bash

2. Visit any website
3. Click "Scan This Page"
4. See result with confidence bar
```
